# Argument Strength Analyzer & Advisor

An AI-powered tool that analyzes arguments sentence by sentence, providing actionable feedback to make your arguments stronger, clearer, and more persuasive.

## Features

- **Sentence-by-Sentence Analysis**: Classifies each sentence as Strong, Weak, Biased, or Logical
- **Actionable Suggestions**: Get tailored improvement suggestions for each argument type
- **Visual Analytics**: Color-coded labels and interactive charts show argument structure
- **Multiple Argument Types**: Supports history-based, science-based, personal experience, and more
- **Export Functionality**: Download analysis results as CSV
